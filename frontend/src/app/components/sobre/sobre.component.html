Sobre o Projeto

Este projeto tem como objetivo coletar, processar e analisar postagens de redes sociais para monitorar a opinião pública sobre serviços públicos. Utilizamos técnicas de scraping para extrair dados, que são processados e armazenados em um banco de dados MongoDB. O back-end é desenvolvido em Spring Boot e o front-end em Angular, proporcionando uma interface moderna e interativa.
